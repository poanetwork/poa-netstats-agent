    poa_agent v0.1.0 POAAgent.Plugins.Collectors.Eth.Stats View Source

    This is a Collector’s Plugin which makes requests to a Ethereum node in order to know if the stats has been changed.

    This Collector needs the url of the node to iteract. That url must be placed in the args field in the config file. For example:

    {:eth_stats, POAAgent.Plugins.Collectors.Eth.Stats, 5000, :eth_stats, [url: "http://localhost:8545"]}

    In this example, the Collector will check with the Ethereum node every 5 seconds if the stats of the node has changed. If that is the case it will send it to the Transfers encapsulated in a POAAgent.Entity.Ethereum.Statistics struct
